# Hirenix AI 🚀

### Powered Interview Intelligence Platform

Hirenix AI is an **AI-powered interview simulation and evaluation platform** designed to help candidates prepare for job interviews and assist recruiters in analyzing candidate performance.

The platform provides a **realistic interview environment**, automated evaluation, performance analytics, and recruiter dashboards — all inside a modern web interface.

---

## 🌟 Features

### 👨‍💻 Candidate Side

* AI-driven mock interviews
* Real-time interview question simulation
* Answer evaluation and scoring
* Skill breakdown and analytics
* Interview performance reports
* Personal dashboard with statistics

### 🧑‍💼 Recruiter Side

* Candidate leaderboard
* Candidate performance comparison
* Skill analytics and ranking
* Interview report viewing
* Recruitment insights dashboard

### 🎨 UI/UX

* Modern responsive design
* Smooth animations and transitions
* Interactive dashboard
* Clean interview interface
* Mobile-friendly layout

---

## 🧠 How It Works

1. User signs up as **Candidate or Recruiter**
2. Candidate starts an **AI mock interview**
3. Candidate answers questions within a timed environment
4. System evaluates responses
5. Scores and analytics are generated
6. Recruiters can review candidate performance
